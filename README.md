NPMStart
=========

NPMStart is a community project.

Version
----

0.0.1

Installation
--------------

```sh
git clone [git-repo-url] npmstart
cd npmstart
npm install
bower install
```

Compile with
--------------

Grunt
```sh
grunt cleanup
grunt default
```


Server
--------------

By default server run on port 3001 or port defined in environmetn variable PORT

http://localhost:3001/

```sh
node ./bin/www
```
